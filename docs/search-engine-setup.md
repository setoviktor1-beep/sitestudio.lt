# Paieškos sistemų išoriniai veiksmai

Produkcijos workflow po sėkmingo GitHub Pages diegimo siunčia pasikeitusių
SiteStudio URL sąrašą į IndexNow. Viešas patvirtinimo raktas laikomas domeno
šaknyje; joks slaptas raktas tam nereikalingas.

Svetainės savininko prieigos vis dar reikalingos šiems veiksmams:

1. Patvirtinti `https://sitestudio.lt/` nuosavybę Google Search Console.
2. Pateikti `https://sitestudio.lt/sitemap.xml` Google Search Console.
3. Patvirtinti svetainę Bing Webmaster Tools ir pateikti tą patį sitemap.
4. Stebėti indeksavimo, Core Web Vitals ir struktūrinių duomenų ataskaitas.

Repozitorijoje nelaikomi paieškos sistemų paskyrų slapti duomenys ar
nepatvirtinti verifikavimo meta elementai.
